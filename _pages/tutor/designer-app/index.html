Form Designer App Features

Build powerful, customized forms with ease! Design forms, add logic, and automate tasks. Control access and gather data efficiently.

Key Features

  • Create Reusable Questions with Validations
  • Create Reusable Forms and Preview them
  • Format Forms with Text and Page Breaks
  • Reorder Questions
  • Create Conditional Logic
  • Publish Forms with Access Control (Employees, Visitors, Public)
  • Create Visitor Groups
  • Test Published Forms
  • Create Form Actions (Evaluate Responses, Trigger Events)
  • Create Form Flows (Multi-Form Tasks)
  • Publish and Test Form Flows
  • Create Workflows for Recurrent Tasks
  • Pause/Resume Workflows
