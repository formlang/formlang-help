Form Actions: Automate tasks based on responses

With form actions, you can automate assessing the form responses and performing other related tasks based on the assessment results.

For example, you can use the form actions to:

  • Calculate the total points for the responses based on the evaluation rules you define.
  • Decide if a respondent passes a screening test or not based on the evaluation rules you define.
  • Send different messages back to respondents based on their responses.
  • Redirect the respondents to new forms based on their responses.
  • Trigger your custom events and then send out notifications based on the subscriptions for the custom events.

Learn about the form action workspace and its general principles. Find specific examples in the links at the end.

The form action workspace is a one-stop shop for creating and editing form actions. To access the workspace, go to Create a Form Action in the {{ site.designApp }}.

As shown in Figure {{ figure_counter }}, the workspace consists of the following key features:

Figure {{ figure_counter }}: Form action workspace
1. Toolbar
The toolbar at the top of the workspace has buttons for opening/creating a form action, saving the form action, and redoing/undoing recent operations.
2. Form action title
The form action title is displayed below the toolbar. Click the title to edit it.
3. Node toolbox
The left node toolbox provides five types of nodes:
  • Form node: Represents a form.
  • Evaluation node:Evaluates the form responses based on the evaluation rules you define.
  • Decision node:Makes a decision based on the results of the evaluation.
  • Feedback node:Sends a message or a new form link to the user.
  • Custom event node: Triggers a custom event.
To create a node, drag it from the toolbox and drop it on the canvas. For more information about node types, see Form Action Nodes: link to documentation.
4. Canvas
The canvas is the main workspace where you create and edit your form actions.
3. Property inspector
The property inspector allows you to edit the properties of the selected edge or node in the canvas.
