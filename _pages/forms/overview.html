Form Design

{{ site.formLangBrand }} is a complete form solution that helps you create, publish, collect, and report on forms for your business. The figure below shows what FormLang can do. We'll describe each feature in more detail below, and you can learn more by clicking the links in the description.

Creating forms

Reusable and clonable forms
Create reusable forms, or clone existing ones and modify them. Save time and effort!
Form versioning
Create new versions of your forms while keeping the old ones. Track changes and manage them easily!
Edit-preview design cycle
Preview changes to your form as you make them, so you can design the form you need.

Creating questions

Reusable and clonable questions
Create reusable questions for different forms, or clone existing ones and modify them. Save time and effort!
Multi-language support
Translate your questions into multiple languages to reach a wider audience.
Standard input validation
Set validation rules for your questions, such as requiring a response or only accepting dates in a certain range.
Single-input questions
Create a variety of question types, including text, email, telephone, password, date, number, checkbox, single-select, and drop-down multi-choice.
Group questions
Group single-input questions together to create a group question, such as a contact card or address card.
Table questions
Create a table question to collect a list of items, such as a list of contacts.
