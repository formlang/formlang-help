Get the Most Out of FormLang: Customize FormLang to your business needs

{{ site.formLangBrand }} allows you to customize the application to meet your specific business needs. With the system settings feature, you can:

  • Use your own organization logo instead of the FormLang logo.
  • Change how your employees register for the application, such as disabling new user registration.
  • Enable or disable the session timeout feature.
  • Update the password complexity requirements.
  • Enable or disable two-factor authentication.
  • Switch to use your own email SMTP service.
  • And more!

For a detailed explanation of the system settings and how to use them, please read the following section.

How to Use the System Settings

To access the system settings feature, go to Settings in the {{ site.formLangBrand }} Portal application. The system settings are grouped into the following 4 tabs:

Appearance
Here you can upload the logo of your organization, to display your organization's logo for your employees on many pages, including the login page and the web applications.
User management
Here you can make changes about how your employees register for FormLang and use it. You can:
  • Disable new user registration.
  • Protect the organization from spam and password decryption on login/registration.
  • Enable the session timeout feature.
  • And more!
Security
Here you can:
  • Update the password complexity requirements.
  • Enable or disable two-factor authentication.
  • Enable or disable account lockout feature for too many login failure attempts.
Email(Smtp)
With this tab, you can switch to use your own email SMTP service.

Additional tips

Here are some tips for making the right changes to the system settings:

  • If you have a large number of employees, you may want to disable new user registration and require all new employees to be added to the system by an administrator.
  • If you want to make sure that your employees' passwords are strong, you can increase the password complexity requirements.
  • If you want to add an extra layer of security to your {{ site.formLangBrand }} account, you can enable two-factor authentication.

If you are unsure about how to make a particular change, please contact {{ site.formLangBrand}} support for assistance.
